# Book Library Collection

A React app showcasing a collection of books using reusable components, built with Vite.

## Features
- Responsive book card components
- Collection summary with statistics
- Modern and clean UI design
- Mobile-friendly layout

## Components
- **BookCard**: Reusable component to display individual book information
- **App**: Main component containing the book collection and layout

## Technologies Used
- React
- Vite
- CSS3 Grid and Flexbox
- JavaScript ES6+
